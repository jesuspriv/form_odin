# Chunin Exams - Sign Up Form

Practice project for **The Odin Project**, focused on building a sign-up form using semantic HTML and pure CSS (no frameworks).

## Description

A themed sign-up form ("Chunin Trials"), inspired by a mix of Norse mythology and the anime *Naruto*. The main goal of this exercise was to practice:

- Semantic form structure (`form`, `label`, `input`)
- Native HTML validation (`required`, `type="email"`, `type="password"`, etc.)
- Layout with **Flexbox** and **CSS Grid**
- Two-column layout (sidebar + content)
- Input styling (`:hover` and `:focus` states)
- Pseudo-classes and simple transitions
- Custom typography with `@font-face`

## Technologies used

- HTML5
- CSS3 (Flexbox, Grid, manual color variables)

## Project structure

```
├── index.html       # Form structure
├── styles.css        # Project styles
```
